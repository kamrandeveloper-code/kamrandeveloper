## Multi-Vendor Ecommerce Platform

- Industry: E-Commerce
- Problem: Business needed multiple independent vendors on one platform
  with separate management but a unified customer experience.
- Solution: Marketplace with separate buyer, seller, and admin portals;
  vendor isolation via role-based access control; unified storefront for
  customers; multi-vendor order splitting and fulfillment tracking.
- Result: Launched with 12 active vendors, processed 500+ orders in the
  first month, average page load under 1.2 seconds.
- Stack: Next.js (with ISR for product pages), .NET Web API (clean
  architecture), PostgreSQL, TypeScript, Tailwind CSS.
- Full case study: https://www.kamrandeveloper.com/case-studies/multi-vendor-ecommerce
