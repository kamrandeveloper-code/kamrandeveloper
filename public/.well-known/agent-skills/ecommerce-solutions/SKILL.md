# E-Commerce Solutions

Kamran builds e-commerce platforms from the ground up — single-vendor
stores through to multi-vendor marketplaces with separate vendor and admin
dashboards, full product catalog management, cart/checkout, and payment
processing.

This is custom platform development. It does not cover third-party
marketplace seller-account or dropshipping setup (e.g. Etsy, eBay, Amazon,
Walmart) — that is not a service currently documented on this site.

## Stack

Next.js, .NET Web API, PostgreSQL, Stripe.

## Delivered example

See `references/case-studies.md` for the multi-vendor marketplace case
study: launched with 12 active vendors, 500+ orders in the first month,
sub-1.2s median page load.

## Engagement

Fixed-scope project, typically 5–12 weeks; multi-vendor builds are scoped
after a detailed discovery call.
