# Admin Dashboards

Kamran builds internal operational dashboards that consolidate a client's
existing data sources into a single, role-based interface — real-time
reporting and operational controls, not a wall of static charts nobody
looks at.

## Stack

Next.js, React, .NET API, PostgreSQL.

## Engagement

Fixed-scope project, typically 3–6 weeks, delivered iteratively — core
dashboard first, then additional views by role.
