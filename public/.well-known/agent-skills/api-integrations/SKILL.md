# API Integrations

Kamran designs and builds the API layer that connects a client's existing
systems — payment processors, CRMs, logistics services — with proper
authentication, error handling, retry logic, and logging so failures are
visible instead of silent.

## Stack

ASP.NET Core, REST, Webhooks, JWT. Payment gateways integrated to date:
Stripe, JazzCash, EasyPaisa, PayPal.

## Engagement

Priced per integration, typically 1–3 weeks each; each integration is
scoped and priced independently.
